What is AquaFeed?

AquaFeed is an end-to-end connected feeding platform with hardware, firmware, and web-based control software. Each feeder can receive commands, report status, and integrate into a central host system for scheduling, monitoring, and management.

How It Works

  1. Connect the feeder A new feeder joins the network and pairs with the host.
  2. Configure schedules Users set feeding times and parameters through the web interface.
  3. Execute feed commands The feeder receives commands and operates the motor locally.
  4. Report back The device sends status, heartbeat, and error information to the host.
